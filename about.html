Welcome to Isekai Legends, an anime fan website dedicated to some of the most popular isekai series ever created. The purpose of this website is to provide anime fans with information about: • That Time I Got Reincarnated as a Slime • Overlord • Arifureta • Mushoku Tensei • Seirei Gensouki Visitors can explore character profiles, story arcs, power systems, and anime statistics while interacting with quizzes and other features. This website was developed as part of a Web Design project using HTML, CSS, and JavaScript and is hosted using GitHub Pages. Our goal is to create an engaging experience for anime fans and help users discover new series based on their interests.
